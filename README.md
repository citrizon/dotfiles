# Citrizon's dotFiles!~ (Monika branch)
This is a repository of my dotfiles seperated by branches. You can check out the branches for my dotfiles for specific themes. Each branch has a setup script that has the same options, certain branch setups can have different settings.

You are currently in the `Monika Branch`

## Setup
First, clone this repository by executing this command:
```
git clone -b ddlc-monika https://github.com/citrizon/dotfiles
```
Second, change your working directory to the repository by executing this command:
```
cd dotfiles
```
After that, you can finally view the options by executing this command:
```
./setup -h
```
